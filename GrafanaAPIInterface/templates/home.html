SAO Engineering Database Plotter

This tool uses Grafana to plot data from the engineering database

Grafana Home

Grafana stores graphs in dashboards which can be saved and viewed any time

Create Dash

- Creates permenant dashboards to be viewed later

Delete Dash

- Deletes a permenant dashboard

Temp Graphs

- Creates graphs in a temporary dashboard that will be deleted after 24 hours. Temporary graphs can be saved to permenant dashboards.
